# Curso ASP.NET Core

> Dudas y consultas: enrique.urra@gmail.com

* [Proyecto01](/Proyecto01): Proyecto para implementar un mantenedor simple de funcionarios. Usa persistencia en memoria (Servicio Cache) y arquitectura basada en Razor pages (NO MVC).
* [Proyecto02](/Proyecto02): Proyecto para implementar un workflow simple de compra de equipamiento. Usa persistencia en memoria (Servicio Cache) y arquitectura MVC.
* [Proyecto03](/Proyecto03): Proyecto para implementar un mantenedor de ventas de servicios, para lo cual se usan datos similares a los de la [página CCHEN](https://www.cchen.cl/?page_id=348). Usa persistencia en base de datos SQL Server Express LocalDB (Entity Framework) y arquitectura MVC.
* [Proyecto04](/Proyecto04): Proyecto para mostrar funcionamiento básico de Identity (scaffolding, login, configuraciones, mantención de usuarios y roles, acceso delimitado por roles). También usa MVC.
* [Proyecto05](/Proyecto05): Proyecto para mostrar funcionamiento básico de Web API, incluyendo interfaz swagger. Utiliza modelos de Proyecto03 y un frontend básico implementado con React-Vite para consumir la API.
